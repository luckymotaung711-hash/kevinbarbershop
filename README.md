# kevinbarbershop

Clients who want to book for a hair cut can just whatsapp us.
Clients can find us through phone numbers, whatsapp numbers, or email.

Menu accessability:Clients can  overview the complete menu online instead of coming to the physical store.

Online Ordering Capability:
The website enables clients to place bookings through whatsapp numbers so that when they come they'll not wait a long que.
